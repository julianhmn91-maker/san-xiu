# San Xiu — Das Buch

PDF-Generator für das Shaolin-Trainingsbuch.

## Lokal ausführen

```bash
pip install weasyprint pypdf
python buch.py
```

Das PDF wird als `San_Xiu_Das_Buch.pdf` gespeichert.

## Via GitHub Actions

Nach dem Push auf `main` wird das PDF automatisch gerendert.  
Download unter **Actions → Render San Xiu PDF → Artifacts → San_Xiu_Das_Buch**.
